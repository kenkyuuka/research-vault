KiriKiri (吉里吉里) is a very popular open source VN engine created by W.Dee.

KiriKiri is used in thousands of games. Assets are stored in XP3 archives, and scripts (`.ks`) are in a human-readable format.

# Identifying

Game data is usually stored in `.xp3` archives, though it may be embedded in the executable as well. [TyranoBuilder](engines/TyranoBuilder.md) games also use `.xp3` archives.

# XP3 archives

XP3 is a fairly simple archive format, [described as a 'virtual file system'](https://github.com/krkrz/krkr2/blob/master/kirikiri2/trunk/kirikiri2/src/core/base/XP3Archive.cpp).
